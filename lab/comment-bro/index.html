    How it works

    What you're doing here

    Paste any post, caption, or thread — the kind of thing you feel socially obligated to comment on but have absolutely nothing to say about. Hit Write my comment. You get a ready-to-paste comment for LinkedIn, X, and Instagram, each leaning into that platform's very specific brand of cliché.

    The honesty slider

    One slider rules all three tabs. Slide left for the most generic, soul-less, engagement-bait comment imaginable — the “This 👏 is 👏 gold” energy. Slide right and it stays funny but starts gently roasting the post itself: the humble-brag, the obvious template, the thing nobody's polite enough to say. It updates in real time, no reloading.

    The bro tokens thing

    This runs on real API calls (they cost money). So each browser gets 5 bro tokens per rolling 12 hours — five full comment sets. The counter at the top of the input field shows how many you have left. Moving the slider is free — that's all generated in one go.

    Hit zero? Wait until your oldest run falls outside the 12-hour window, or — radical idea — leave an actual comment yourself.

    Be cool about it

    Comments are meant to be funny, not mean. Even the highest honesty level roasts the genre, not the person. Read it before you post it — you're the one whose name is on it.
